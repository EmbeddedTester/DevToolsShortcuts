Dev Tools Shortcuts

You can find all shortcuts here: https://developer.chrome.com/devtools/docs/shortcuts

Shortcuts Mac

Open Developer Tools                        Cmd + Opt + I
Refresh the page                            Cmd + R
Text search within current file or panel    Cmd + F
Text search across all sources              Cmd + Opt + F
Undo change                                 Cmd + Z
Save changes to local modifications         Cmd + S
Go to line                                  Cmd + G
Search by filename                          Cmd + O
Go to member                                Cmd + Shift + O

Shortcuts Windows

Open Developer Tools                        F12, Ctrl + Shift+ I
Refresh the page                            F5, Ctrl + R
Text search within current file or panel    Ctrl + F
Text search across all sources              Ctrl + Shift + F
Undo change                                 Ctrl + Z
Save changes to local modifications         Ctrl + S
Go to line                                  Ctrl + G
Search by filename                          Ctrl + O
Go to member                                Ctrl + Shift + O
